# 公网发布

## 目标
- 让手机或其他网络环境下的浏览器，直接通过公网链接访问当前股票网站。

## 推荐顺序
1. 先用临时公网链接做验证。
2. 确认页面和数据接口稳定后，再做长期部署。

## 方案A：临时公网链接
- 适合今天马上测试手机能不能打开。
- 电脑需要保持开机，并且本地服务保持运行。
- 链接通常是临时的，重启后可能变化。

### 推荐工具
- Cloudflare Quick Tunnel

### 基本步骤
1. 先安装项目依赖：`pip3 install -r requirements.txt`
2. 启动本地服务：`python3 server.py`
3. 再启动临时隧道，让公网请求转发到本地 `8000` 端口。
4. 把生成的公网链接发到手机浏览器打开。

## 方案B：长期稳定链接
- 适合后续真正给别人访问。
- 适合继续扩展全A股、筛选、收藏和后续登录体系。
- 部署后不需要你的电脑一直开着。

### 推荐平台
- Render

### 适合当前项目的原因
- 支持 Python Web 服务。
- 有固定公网域名。
- 对首版验证比较友好。

### 部署前项目要求
- 服务必须监听 `0.0.0.0`
- 端口必须兼容平台提供的 `PORT` 环境变量
- 依赖通过 `requirements.txt` 安装

## 当前项目已满足
- `server.py` 已默认监听 `0.0.0.0`
- `server.py` 已兼容 `PORT`
- 静态页面和接口由同一个 Python 服务提供

## 本阶段建议
- 本阶段先做“临时公网链接”验证。
- 如果手机确认能打开，再进入正式部署。
