# 技术方案

## 阶段1
- 首页展示全A股股票列表。
- 搜索框支持按股票代码回车直达详情页。
- 详情页展示成交价、市值、毛利率、市盈率四张卡片。
- 数据源优先使用免费抓取方案。
- 首版采用 Python HTTP 服务 + AkShare 数据适配层。
- 前端以轮询方式每 5 秒刷新一次。
- 服务默认监听 `0.0.0.0:8000`，支持同一局域网内其他设备访问。

## 数据策略
- 最新价、市值：按5秒刷新。
- 毛利率：按财报数据更新。
- 市盈率：按近五年收盘价与当前动态市盈率估算历史走势。
- 详情页走势图改为五年日线序列，横轴按每日收盘日期展示。
- 股票列表的毛利率来自最新财报报表并与实时行情合并。

## 风险控制
- 数据源要可替换。
- 抓取失败要有降级提示。
- 不先做复杂登录和同步。
- 当前环境无法完整联通真实外部接口，需在真实桌面网络下做最终联调。
